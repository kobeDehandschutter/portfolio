Java
In het eerste jaar van deze bachelor heb ik in het tweede semester een Java-project gemaakt in een groepje van vier personen. De opdracht was om het bekende gezelschapspel Rummikub online na te maken, gebruik makende van Java en JavaFX. Hieruit heb ik veel geleerd, niet alleen over Java en JavaFX, maar ook over teamwerk, dit verliep in het begin nogal stroef. Dit komt ook omdat alles online moest gebeuren en er dus geen mogelijkheid was om mijn teamgenoten beter te leren kennen, wat heel jammer was natuurlijk.
